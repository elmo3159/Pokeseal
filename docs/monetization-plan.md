# ポケシル 課金システム仕様書

## 1. 概要

ポケシルの課金システムは、子ども向けアプリとして以下の原則に基づいて設計されています：

- **公平性**: 無課金でも十分楽しめる
- **透明性**: 何が得られるか明確に表示
- **安全性**: 過度な課金を防ぐ仕組み
- **価値提供**: 課金に見合った価値を提供

---

## 2. 通貨システム

### 2.1 通貨の種類

| 名称 | アイコン | 用途 | 獲得方法 |
|------|---------|------|----------|
| シルチケ | 🎫 | 通常ガチャ | デイリーボーナス、ミッション、広告視聴 |
| プレシル | 💎 | プレミアムガチャ | サブスク特典、イベント報酬 |
| どろっぷ | 💧 | ガチャ・アイテム購入 | 課金購入 |

**消費優先度:** シルチケ/プレシル → どろっぷ（チケット不足時にどろっぷ使用を案内）

### 2.2 どろっぷパック（有料通貨）

ガチャを回したり、特別なアイテムを購入するための有料通貨。

| パック名 | 価格 | どろっぷ数 | ボーナス | 1どろっぷあたり |
|----------|------|----------|----------|----------------|
| おためし | ¥120 | 120 | - | ¥1.00 |
| ちょっと | ¥370 | 400 | +8% | ¥0.93 |
| おこづかい | ¥730 | 850 | +16% | ¥0.86 |
| たっぷり | ¥1,480 | 1,800 | +22% | ¥0.82 |
| もりもり | ¥3,680 | 4,800 | +30% | ¥0.77 |
| スペシャル | ¥6,100 | 8,500 | +39% | ¥0.72 |

**初回購入ボーナス**: 初めての購入時は獲得どろっぷ2倍

### 2.3 シルチケ（通常ガチャ用チケット）

通常ガチャを1回引くためのチケット。以下の方法で獲得可能：

| 獲得方法 | 枚数/日 | 条件 |
|----------|---------|------|
| デイリーボーナス | 3枚 | ログイン時に自動配布 |
| デイリーミッション | 2枚 | ミッションクリア |
| 広告視聴 | 最大10枚 | 広告1回につき1枚 |
| **合計（無課金）** | **15枚** | - |

### 2.4 プレシル（プレミアムガチャ用チケット）

プレミアムガチャ専用のチケット。以下の方法で獲得可能：

| 獲得方法 | 枚数 | 条件 |
|----------|------|------|
| サブスク特典 | 週1〜3枚 | パスプラス以上 |
| イベント報酬 | 不定 | イベント参加 |
| 特別キャンペーン | 不定 | 期間限定 |

## 5. デイリーミッション

毎日リセットされるミッション。クリアでシルチケ獲得。

| ミッション | 報酬 |
|------------|------|
| タイムラインにコメントする | シルチケ1枚 |
| シールを1枚はる | シルチケ1枚 |
| **合計** | **シルチケ2枚** |

---

## 6. 広告システム

### 6.1 リワード広告

- 1日最大10回視聴可能
- 1回視聴につきシルチケ1枚
- 広告の長さ: 15〜30秒
- スキップ不可（リワード獲得条件）

### 6.2 広告スキップ（パスプラス以上）

- パスプラス/デラックス会員は広告視聴不要
- 毎日ログイン時に自動で+10回分のシルチケ付与

## 7. UI/UX設計

### 7.1 アクセスポイント

| 場所 | 導線 |
|------|------|
| ガチャ画面 | シルチケ/どろっぷ不足時にショップへ |

### 7.2 購入フロー

```
シルチケ/どろっぷ不足
    ↓
「もっとほしい？」ダイアログ
    ↓
┌─────────────────────────────┐
│  選択肢:                      │

│  4. パスに入る                  │
└─────────────────────────────┘
    ↓
各モーダルへ遷移
```

- 平易な日本語を使用

- 購入後は「ありがとう！」メッセージ

## 9. 技術実装メモ

### 9.1 状態管理

```typescript
interface UserMonetization {
  // 通貨
  tickets: number   // シルチケ
  gems: number      // プレシル（プレミアムシルチケ）
  stars: number     // どろっぷ

  // サブスク
  subscription: 'none' | 'light' | 'plus' | 'deluxe'
  subscriptionExpiresAt: string | null

  // デイリー
  dailyTicketsCollected: boolean
  dailyStarsCollected: boolean
  dailyMissionsCompleted: string[]
  adsWatchedToday: number

  // 統計
  totalStarsPurchased: number
  isFirstPurchase: boolean
}
```

- iOS: StoreKit 2
